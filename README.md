# FUTURE_PE_02
UGC Ad Content Pack for Local Businesses

Project Title

UGC Ad Content Pack for Local Businesses

Project Description

This project demonstrates the use of Artificial Intelligence (AI) and Prompt Engineering to generate high-quality User-Generated Content (UGC) advertisements for local businesses. The framework is designed to create engaging hooks, authentic UGC ad scripts, and persuasive call-to-action (CTA) content for social media platforms such as Instagram Reels and YouTube Shorts.

The goal of this project is to help local businesses create effective marketing content quickly and efficiently using AI.

Business Selected

Business Name: Shyam Institute
Business Type: Coaching Institute
Target Audience: Job Aspirants and Degree Students

Tool Used

- ChatGPT (AI Content Generation)
- Prompt Engineering Framework

Objective

To build a reusable AI prompt framework that can:

- Generate attention-grabbing hooks
- Create authentic UGC-style ad scripts
- Write persuasive Call-to-Action (CTA) lines
- Adapt content for different local businesses
- Maintain a consistent brand voice
- Produce social media-ready marketing content

Prompt Framework Structure

The framework accepts the following inputs:

- Business Name
- Business Type
- Target Audience
- Platform
- Tone of Voice

Tone Used

- Authentic
- Friendly
- Human
- Non-Salesy

Prompt Logic

The AI follows a structured workflow:

1. Analyze the business type.
2. Understand the target audience.
3. Identify customer pain points and goals.
4. Generate engaging hooks.
5. Create UGC ad scripts using the Hook → Problem → Solution → CTA framework.
6. Write persuasive CTA lines.
7. Ensure the content is platform-ready for Instagram Reels and YouTube Shorts.
